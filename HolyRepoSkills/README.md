## üïπÔ∏è Controls & Hotkey Configuration

You can customize the hotkeys for opening the skill page and activating your equipped skill through the mod‚Äôs config file.

> ‚ö†Ô∏è **Important:** Make sure to **launch the game at least once** before editing the config file ‚Äî this ensures it's generated properly.

### üîß Changing Hotkeys via Thunderstore
1. Go to the **Thunderstore Mod Manager**.
2. Select your profile and click **Edit Config** on the left panel.
3. Navigate to: `BepInEx/Config/JP_RepoHolySkills.cfg`.
4. Click **Edit Config**.
5. Update the following keybinds as desired:
   - `OpenSkillSelectionPage`
   - `ActivateSkill`

### üß© Changing Hotkeys Manually
1. Go to your `BepInEx/config` folder.
2. Open the file: `JP_RepoHolySkills.cfg`.
3. Find and edit these settings:
   - `OpenSkillSelectionPage`
   - `ActivateSkill`
